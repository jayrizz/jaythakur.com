# jaythakur.com

AI Portfolio - Timeline-first design showcasing human-AI collaboration.

## Quick Start

```bash
cd ~/openclaw-workspace/jaythakur.com
npm install
npm run dev
```

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel
3. Connect domain jaythakur.com

## Tech Stack

- Next.js 14 (React)
- Framer Motion (animations)
- TypeScript
- Vercel (hosting)

## Design

- Dark theme default
- Timeline with scroll animations
- Mobile-first responsive
- Orange accent color (#ff6b35)
